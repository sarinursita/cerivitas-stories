# Cerivitas Stories 📚

Kumpulan **Choose Your Own Adventure (CYOA)** games interaktif untuk anak-anak.
Dibuat untuk workshop **AI Agent for Kids** oleh Cerivitas.

| Petualangan Ajaib | Petualangan fantasi | 5-10 menit | [Main](https://sarinursita.github.io/cerivitas-stories/games/cyoa-adventure.html) |
| Misteri Pensil Ajaib | Detektif/Misteri | 5-10 menit | [Main](https://sarinursita.github.io/cerivitas-stories/games/cyoa-misteri.html) |

## 🔧 Cara Pakai

Buka link di atas di browser, langsung main! ✅

## 📝 Spesifikasi

- **Usia:** 6-12 tahun (versi pendek), 9-12 tahun (versi panjang)
- **Konten:** 100% aman, no romance, no horror
- **Bahasa:** Indonesia
- **Teknologi:** HTML + CSS + JavaScript (single file)
- **Hosting:** GitHub Pages
