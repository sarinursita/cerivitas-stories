## Spesifikasi

- Usia: 6-12 tahun (versi pendek), 9-12 tahun (versi panjang)
- Konten: 100% aman, no romance, no horror
- Bahasa: Indonesia
- Teknologi: HTML + CSS + JavaScript (single file)
- Hosting: GitHub Pages

Dibuat oleh Cleo AI & Shabira untuk Workshop Cerivitas.
